# Goal
1. To train a diffusion model that takes $I(t)$ and random number and stochastically predicts the corresponding parameters set $\theta$.
2. To train a diffusion model that takes $I(t)$, $\theta_{condition}\,\,(h_0, f_0, t_s)$ and random number and stochastically predicts the corresponding parameters set $\theta_{inference}\,\,(J_e, b_1, b_2)$.
