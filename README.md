# Tearfilm_SBI_diffusion_ODE
Repository for SBI style inverse problem for the tear film ODE based on diffusion model.
# Resources
Tearfilm ODE dataset (link: https://drive.google.com/file/d/1Pk22KO7xq5RaqP7kE5dCHTr4lkzvqq72/view?usp=drive_link)
# Goal
To train a diffusion model that takes I(t) and random number and stochastically predicts the corresponding parameters set $\theta$.
